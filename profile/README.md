# Financian Media Platform

Financian Media is a modular SaaS platform for content management and media processing.

## Platform Services

- **financian-server** — Backend API
- **financian-client** — User-facing web application
- **financian-admin** — Admin dashboard
- **financian-media-service** — Image processing and uploads
- **financian-bootstrap** — Local development environment
- **financian-packages** — Shared UI and API contracts

## Tech Stack

- Node.js
- Express.js
- TypeScript
- Next.js
- Python
- PostgreSQL
